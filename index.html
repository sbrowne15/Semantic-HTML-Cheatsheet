Guide to Semantic HTML Tags

Moving away from the <div> tag can be a little intimidating when first learning HTML, but writing semantic code makes work infinitely easier to both read and style with CSS. The table provided below is an attempt to keep tags straight for those new to HTML or as a quick reference for anyone that might need it.

Tag Name Tag Open Tag Close Description
Header <header> </header> Describes Content at Top of Page Body
Nav <nav> </nav> Often in Header and Denotes Page's Navigational Links
Main <main> </main> Placed Around the Main Content of Page
Footer <footer> </footer> Denotes Footer Content at Page Bottom
Video <video> </video> Allows the Addition of Video to a Page, Needs src Attribute
Audio <audio> </audio> Allows Audio Files to be Added to Page, Needs src Attribute
Embed <embed> Self-Closing Adds Any Type of Media to Page, Needs src Attribute
Figure <figure> </figure> Marks Media like Images, Diagrams, or Code Snippets
Figcaption <figcaption> </figcaption> Exists Within Figure Tag to Describe Figure
Section <section> </section> Defines Elements of the Page like Chapters or Headings
Article <article> </article> Marks Content that Stands on Its Own, i.e. Blogs or Comments
Aside <aside> </aside> Holds Bonus Information that Enhances Other Elements like Quotes
